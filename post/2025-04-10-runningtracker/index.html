Opportunity to make some nice tests

 Posted on April 10, 2025  |  1 minutes  |  99 words  |  Paolo Fullone

This project is a RESTful API to keep the track of running amateur activities.

I’ve did it since I am an amateur runner.

So, the API itself was not a work challenge, it was more a fun project to do in my free time. But I also took the opportunity to make all tests I’m aware of and I left a detailed explanation of each one of them in the readme of the project in github.

If you have something to share or discuss about the project, please feel free to reach out and we can have a chat.
